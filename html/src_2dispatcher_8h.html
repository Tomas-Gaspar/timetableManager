Invokes the method of a class X whose member pointer type is defined by W<X>::method where X matches the type of the object held by v that is stored in a variant V.

Template Parameters
Wa template class where W<X>::method points to a method of class X one of the types of V.
Vthe type of the variant
Asthe parameter types used to invoke the method
Parameters
va object of the variant type for which to invoke the method
argsthe parameters to pass to the method
Note
This implementation assumes a maximum of 10 types in the variant (easy to extend though by adding more cases).
