)
private

Appends a vector of Strings to another vector of Strings.

Parameters
[in]vectorThe vector of Strings to be appended to.
[in]itemsThe potential selection of Strings to be appended.
[in]maskA mask that designates which elements from items will be appended. If not present then all elements from items will be appended. If present must be the same size as items.

Note that items is left unchanged.
