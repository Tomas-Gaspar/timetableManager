Shrinks a vector of Strings by removing items from the end.

Parameters
[in]vectorThe vector to expand.
[in]countOptional number of items to shrink by.
If not present, the vector is shrunk by one item. If negative, the vector will grow.

This is perhaps a more performance way of going:

vector = vector(:SIZE(vector)-count)
