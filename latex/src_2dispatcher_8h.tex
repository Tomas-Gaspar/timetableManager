Invokes the method of a class {\ttfamily X} whose member pointer type is defined by {\ttfamily W\texorpdfstring{$<$}{<}X\texorpdfstring{$>$}{>}::method} where {\ttfamily X} matches the type of the object held by {\ttfamily v} that is stored in a variant {\ttfamily V}. 
\begin{DoxyTemplParams}{Template Parameters}
{\em W} & a template class where {\ttfamily W\texorpdfstring{$<$}{<}X\texorpdfstring{$>$}{>}::method} points to a method of class {\ttfamily X} one of the types of {\ttfamily V}. \\
\hline
{\em V} & the type of the variant \\
\hline
{\em As} & the parameter types used to invoke the method \\
\hline
\end{DoxyTemplParams}

\begin{DoxyParams}{Parameters}

\hline
\end{DoxyParams}
\begin{DoxyNote}{Note}
