\doxysubsection{Detailed Description}
\mbox{\hyperlink{class_class}{Class}} representing a token in the compiled regular expression token stream. \mbox{\hyperlink{class_a}{A}} token has a kind and an optional value whose meaning depends on the kind. It is also possible to store a (from,to) character range in a token.
