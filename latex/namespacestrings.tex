\doxysubsection{Detailed Description}
Routines for managing character strings, particularly one dimensional arrays (vectors) of strings. 

The \mbox{\hyperlink{class_string}{String}} type and associated procedures exposed by this module conform to the typical interface for vectors\+:


\begin{DoxyItemize}
\item The item component obtains the item for a particular element in the vector.
\item Append\+: add an object to the vector.
\item Grow\+: create space for new objects at the end of the vector, but do not define those objects.
\item Shrink\+: remove objects from the end of the vector. 
\end{DoxyItemize}
