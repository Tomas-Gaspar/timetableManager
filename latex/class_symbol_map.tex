\doxysubsection{Detailed Description}
\subsubsection*{template$<$class T$>$\newline
class Symbol\+Map$<$ T $>$}
\mbox{\hyperlink{class_class}{Class}} implementing a symbol map that maps symbol names to objects. Symbol names do not have to be unique. Supports adding symbols with \mbox{\hyperlink{class_symbol_map_a5ee5036f5adbd147a02e4aa476fc8b72}{add()}}, removing symbols with \mbox{\hyperlink{class_symbol_map_a36b000d1d74ec4820b0d1c3a57dd645c}{remove()}}, and finding symbols with \mbox{\hyperlink{class_symbol_map_a7f278648058288ad501bdb8d6d10c223}{find()}}.
