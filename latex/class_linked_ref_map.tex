class Linked\+Ref\+Map$<$ T, Hash, Key\+Equal, Map $>$}
Container class representing a vector of objects with keys. 

Objects can be efficiently be looked up given the key. Objects are {\itshape not} owned by the container, the container will only hold references. When adding objects the order of addition is kept, and used while iterating.
