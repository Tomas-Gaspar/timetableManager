\doxysubsection{Detailed Description}
\mbox{\hyperlink{struct_object}{Object}} representing the matching results. It consists of an array of \mbox{\hyperlink{classreg_1_1_sub_match}{Sub\+Match}} objects. The first entry of the array represents the whole match, any next elements represent each of the capture ranges.

For example string {\ttfamily @42} and expression {\ttfamily @(\textbackslash{}\textbackslash{}d+)} will have two Submatches, match\mbox{[}0\mbox{]} will point to the input string as a whole, and match\mbox{[}1\mbox{]} will point to the number 42 only.
