\begin{DoxyReturn}{Returns}
If {\itshape vector} is present, the result size is that of {\itshape vector}, otherwise the result size is the number {\itshape t} of true elements in {\itshape mask}.
\end{DoxyReturn}
Element {\itshape i} of the result is the element of {\itshape array} that corresponds to the {\itshape i}th true element of {\itshape mask}, taking elements in array element order, for {\itshape i} = 1, 2, ..., {\itshape t}. If {\itshape vector} is present and has size {\itshape n} \texorpdfstring{$>$}{>} {\itshape t}, element {\itshape i} of the result has the value {\itshape vector}({\itshape i}), for {\itshape i} = {\itshape t} + 1, ..., {\itshape n}.
