{\footnotesize\ttfamily \mbox{\hyperlink{classvhdl_1_1parser_1_1_token}{Token}} $\ast$ vhdl\+::parser\+::\+Token\+::special\+Token = 0}

This field is used to access special tokens that occur prior to this token, but after the immediately preceding regular (non-\/special) token. If there are no such special tokens, this field is set to NULL. When there are more than one such special token, this field refers to the last of these special tokens, which in turn refers to the next previous special token through its special\+Token field, and so on until the first special token (whose special\+Toke\+\_\+ field is NULL). The next fields of special tokens refer to other special tokens that immediately follow it (without an intervening regular token). If there is no such token, this field is NULL.
