\doxysubsection{Detailed Description}
C-\/like language parser using state-\/based lexical scanning. 

This is the language parser for doxygen. It is somewhat fuzzy and supports C++ and various languages that are closely related to C++, such as \mbox{\hyperlink{class_c}{C}}, \mbox{\hyperlink{class_c}{C}}\#, Objective-\/C, Java, PHP, and IDL.
