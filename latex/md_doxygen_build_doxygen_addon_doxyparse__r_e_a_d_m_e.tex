This directory contains an \char`\"{}source parsing engine\char`\"{} based on doxyapp code.

\mbox{\hyperlink{class_doxyparse}{Doxyparse}} modifies the default output of \mbox{\hyperlink{class_doxygen}{Doxygen}} and dumps the dependencies among code elements in a YAML format, instead of output it in a human-\/readable format, as \mbox{\hyperlink{class_doxygen}{Doxygen}} does \mbox{\hyperlink{class_doxyparse}{Doxyparse}}\textquotesingle{}s output is intended to produce a machine-\/readable output.

\mbox{\hyperlink{class_doxyparse}{Doxyparse}} has been used in many software engineering research (as a source-\/code static analysis tool) regards on software metrics, quality metrics and so on, \mbox{\hyperlink{class_doxyparse}{Doxyparse}} was first used by the \href{http://analizo.org}{\texttt{ Analizo}} toolkit, a suite of source code analysis tools, aimed at being language-\/independent and extensible, able to extract and calculate a fair number of source code metrics, generate dependency graphs, and other software evolution analysis.
