{\footnotesize\ttfamily bool read\+Code\+Fragment (\begin{DoxyParamCaption}\item[{const \mbox{\hyperlink{class_q_c_string}{QCString}} \&}]{file\+Name,  }\item[{int \&}]{start\+Line,  }\item[{int \&}]{end\+Line,  }\item[{\mbox{\hyperlink{class_q_c_string}{QCString}} \&}]{result }\end{DoxyParamCaption})}

Reads a fragment of code from file {\itshape file\+Name} starting at line {\itshape start\+Line} and ending at line {\itshape end\+Line} (inclusive). The fragment is stored in {\itshape result}. If FALSE is returned the code fragment could not be found.

The file is scanned for a opening bracket (\textquotesingle{}\{\textquotesingle{}) from {\itshape start\+Line} onward The line actually containing the bracket is returned via start\+Line. The file is scanned for a closing bracket (\textquotesingle{}\}\textquotesingle{}) from {\itshape end\+Line} backward. The line actually containing the bracket is returned via end\+Line. Note that for VHDL code the bracket search is not done.
