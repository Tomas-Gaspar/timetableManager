\doxysubsection{Detailed Description}
Singleton for configuration variables.

This object holds the global static variables read from a user-\/supplied configuration file. The static member \mbox{\hyperlink{class_config_impl_a501e98a30e96c9930ac4b1791b80c09a}{instance()}} can be used to get a pointer to the one and only instance.
